# Elm Tutorial

Source code for a tutorial on building web applications

### Download Dependencies

```
elm package install
```

### To install the server
```
npm install
```

### To start the server
```
elm-make src/Main.elm --output=app.js
node server.js
```
Server runs at [http:://localhost:3000](http:://localhost:3000)
